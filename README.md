# TODO APP
Contoh aplikasi sederhana untuk personal task management

## Stack
- Node Express v4.^
- MySQL / MariaDB

## Instalasi
1. Clone repo
2. Masuk ke direktori repo
2. Siapkan database
	1. Masuk ke mysql command
	2. `create database simple_todo_app;`
	3. `use simple_todo_app;`
	4. `source databases/simple_todo_app.sql;`
2. `npm install`
3. `npm run dev`
